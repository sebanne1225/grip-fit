Grip Fit
BOOTH 配布パッケージ案内
version: 0.1.0

このツールは、握り拳などのハンドジェスチャー状態での持ち物 Transform 位置合わせを非破壊で保存するツールです。

主な方針:
- このツールは非破壊で動作し、元のアバターには直接変更を加えません
- Play Mode 中に位置合わせした姿勢は、Play Mode を終了するだけで自動的に確定します

最初にやること:
1. GitHub repo の README を確認してください
2. まずは VCC / VPM で導入してください
3. Modular Avatar Bone Proxy を設定済みのギミックに Grip Fit Offset を追加してください

導入の基本:
- 主導線は VCC / VPM です
- Git URL 導入は補助扱いです
